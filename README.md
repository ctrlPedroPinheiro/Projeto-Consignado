Projeto desenvolvido para auxiliar e automatizar a análise dos relatórios de consignados em convênio pela **CAIXA ECONÔMICA FEDERAL**.

## Sobre o Projeto

O sistema foca em processar dados brutos de relatórios bancários para extrair informações relevantes de forma rápida, facilitando a conferência de margens e contratos consignados.

### Funcionalidades principais:
* **Processamento de Dados:** Leitura de arquivos de convênio da CEF.
* **Análise de Margem:** Cálculo e validação automática.
* **Interface:** Visualização simplificada utilizando componentes Web (HTML/CSS).

## Estrutura de Pastas

* `consignados/`: Contém as classes principais em Java.
* `README.md`: Documentação do projeto.
